# Project History

This directory preserves **frozen historical artifacts** from earlier stages of the
*Weigh to Go!* project. These documents are kept for reference and traceability — they
record how the project evolved — and are **not actively maintained**. Current,
authoritative documentation lives elsewhere under `docs/`.
